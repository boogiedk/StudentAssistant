Приветствуем, гость!

Веб-сервис Student Assistant является open source проектом. Разработан с целью предоставить удобный и гибкий способ контроля за учебным расписанием.

Что умеет Student Assistant?

Мы регулярно улучшаем текущий функционал и добавляем новый. Далее указаны краткие описания основных возможностей сервиса.

Данные о дне недели

Сервис предоставляет возможность получить данные о текущем дне, которые необходимы для обучения.

Расписание

Student Assistant предоставляет возможность вести расписание как индивидуальное, так и групповое.

Учёт посещаемости

Наш сайт позволяет контролировать посещаемость учебных предметов, а затем делать на ее основе отчёты.

Какие технологии использует Student Assistant?

Сервис состоит из множества технологий, что позволяет привлечь внимание разных по своим предпочтениям и возможностям разработчиков.

ASP.NET Core

Ключевой частью нашего сервиса является технология от Microsoft - ASP.NET Core 2.0

Angular

Frontend нашего сервиса целиком написан на JavaScript фрейморке - Angular.

TypeScript

В качестве основного языка для клиенской части используется TypeScript.

Azure

Сайт развернут на серверах Azure. Удобный интерфейс позволяет легко администрировать сервис.

TeamCity

В качестве многофункционального сервера непрерывной интеграции используется TeamCity.

Trello

Для контроля процесса разработки используется Agile методология - kanban доска в сервисе Trello.
